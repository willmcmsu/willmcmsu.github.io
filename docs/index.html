AWS Cloud Computing

This project performs an analysis of the NYC 311 service requests to identify patterns and complaint types by agency and borough. The project utilizes AWS services for data storage, processing, and visualization."

SQL AWS S3 AWS Athena AWS Sagemaker pandas numpy Jupyter Notebooks matplotlib sklearn
More details

Problem

The mayor's office wants to know whether certain types of 311 complaints are likely to be resolved within 3 days - can we build a model to flag fast vs. slow resolutions at intake?

Approach

Two main datasets were used: complaints.csv and agencies.csv. The complaints dataset contains 311 service request records, while the agencies dataset provides information about the city agencies responsible for handling those requests. The data was stored in AWS S3 and queried using AWS Athena to perform exploratory data analysis and feature engineering. A machine learning model was then trained using AWS Sagemaker to predict whether a complaint would be resolved within 3 days based on features such as problem, agency, borough, and hour of the day.

Results & Impact

The model achieved an accuracy of 78% on the test set, indicating that it can reasonably predict which complaints are likely to be resolved quickly. This information can help the mayor's office prioritize resources and improve response times for certain types of complaints.

Southwest Lansing Analytics and Mapping

For this project, myself and three classmates teamed up with Michigan State University’s Center for Community and Economic Development (CCED) to analyze and visualize data about the Southwest Lansing community. We created an app that allows users to explore various types of data, from general demographics to specific food access, all within the Southwest Lansing area. The app was designed to be user-friendly and interactive, allowing non-tecnhical users to easily navigate and understand the data.

JavaScript HTML/CSS Python Excel ArcGIS OnRender
More details

Problem

The CCED wanted to team up with MSU data science students in an effort to create a tool that could be used by local organizations to explore and understand the Southwest Lansing community. The goal was to create an interactive app that would allow users to explore various types of data that could be focused around the needs of these local organizations, such as housing data and food access.

Approach

Our team sorted through over 50 data sources to find the most granular and relevant data for the Southwest Lansing area. We used ArcGIS to create maps and visualizations of the data, and then built an interactive web app using JavaScript, HTML/CSS, and Python to allow users to explore the data in a user-friendly way. We also worked closely with the CCED and local organizations to ensure that the app met their needs and was easy to use.

Results & Impact

This app will be used as a foundation for both the CCED, their community partners, and local organizations to explore and understand the Southwest Lansing community. Future groups will be able to follow our structure and see our data catalog, to be able to implement their own data and build on our work. Thank you to the CCED for the opportunity to work on this project and create a tool that truly has the potential to make a positive impact in the community.
